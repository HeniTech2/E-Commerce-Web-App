# 🛒 E-Commerce Web App

A modern full-stack E-Commerce web application built with React, Node.js, Express, Prisma, and PostgreSQL.

---

## 🚀 Features

- 🔐 User authentication (login/register)
- 🛍️ Product listing and details
- 🛒 Shopping cart system
- 📦 Order management
- 👨‍💼 Admin dashboard
- 💳 Payment integration (Chapa)
- 📧 Email notifications
- 📁 Image upload support

---

## 🏗️ Tech Stack

### Frontend
- React (Vite)
- React Router
- Context API
- Tailwind CSS

### Backend
- Node.js
- Express.js
- Prisma ORM
- PostgreSQL

### Others
- JWT Authentication
- Multer (file upload)
- Chapa Payment Gateway
